# Ansible Docker for Wordpress

## Supported instance:

- MariaDB

- PHP(php-fpm)
  
- Nginx

## Usage(Should execute by the following command sequence)

``` shell
mariadb/run.sh
php/run.sh
nginx/run.sh
```
